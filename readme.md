# Mini Moteur de Recommandation (TMDB)

Projet pédagogique : mini moteur de recommandation de films basé sur l’API TMDB.

## Stack

- HTML (single page app via hash routing)
- JS (ES Modules)
- SASS (compilé en CSS)

## Démarrage

Ouvrir simplement `index.html` dans un navigateur pour voir la structure de base.

Les étapes suivantes (connexion à l’API TMDB, récupération et scoring) seront ajoutées progressivement.
